## 🚨 트러블슈팅

### 자주 발생하는 문제
1. **CORS 에러**
   - 백엔드 CORS_ORIGINS 환경 변수 확인
   - 프론트엔드 도메인이 정확히 설정되었는지 확인

2. **데이터베이스 연결 실패**
   - DATABASE_URL 환경 변수 확인
   - PostgreSQL 서비스 상태 확인

3. **빌드 실패**
   - 의존성 버전 충돌 확인
   - 빌드 로그에서 에러 메시지 확인

4. **환경 변수 인식 안됨**
   - 환경 변수 이름 오타 확인
   - 플랫폼별 환경 변수 설정 방법 확인
